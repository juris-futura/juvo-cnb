# juvo-cnb

## How to build

### Builder

1. Install docker, kubectl, and [pack cli](https://buildpacks.io/docs/for-platform-operators/how-to/integrate-ci/pack/)
1. Run:
    ```bash
    cd cnb-components/builders/juvo-builder
    ./make-builder v{version}
    ```
